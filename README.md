# learn-qt-2026

A 14-week, project-based curriculum for learning Qt 6 desktop development. **CMake only, VS Code + clangd, Widgets only** — no Qt Creator, no QML.

## Through-Line Project: Multi-Tab Dev Console

The curriculum is built around one incremental project — a **Multi-Tab Dev Console** with:
- **Log Viewer** — open, parse, filter, and search log files
- **Text Editor** — syntax highlighting, line numbers, find/replace, file browser
- **Serial Monitor** — connect to serial ports, send/receive, hex/ASCII display

Project source lives in [`project/`](project/).
